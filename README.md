# LinkSecure

Secure file sharing platform with analytics, password protection, and role-based access control.

## Getting Started

Prerequisites:
- Node.js and npm

Install dependencies:
```sh
npm install
```

Run the development server:
```sh
npm run dev
```

Build for production:
```sh
npm run build
```

Preview production build:
```sh
npm run preview
```

## Tech Stack
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
